# Sudoku Solver AI

## Overview

Sudoku Solver AI is a C++ application that solves Sudoku puzzles using constraint propagation, logical inference, and search techniques. The solver analyzes candidate values for each cell, applies multiple inference rules, and progressively reduces the search space until a valid solution is found.

The project includes a graphical interface that allows users to visualize puzzle states and observe how different solving strategies contribute to the final solution.

## Features

* Sudoku puzzle loading
* Domain computation for empty cells
* Constraint propagation
* Single Candidate inference
* Naked Subset inference
* Unique Candidate inference
* Search and backtracking
* Interactive GUI visualization
* Step-by-step solving process

## Purpose

This project explores how logical reasoning and constraint satisfaction techniques can be applied to solve complex puzzles efficiently. It demonstrates AI-inspired problem-solving methods without relying on machine learning models.
